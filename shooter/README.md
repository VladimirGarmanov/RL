# RL Shooter — Self-Play

Шутер с видом сверху, где RL-агент учится воевать против себя (self-play). Агент — синий кружок с пушкой, противник (бот) — красный.

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![pygame](https://img.shields.io/badge/pygame-2.x-green) ![SB3](https://img.shields.io/badge/stable--baselines3-PPO-orange)

## Механика

- Персонаж **постоянно вращается**
- Держишь **ПРОБЕЛ** — едет вперёд в зафиксированном направлении, не крутится
- **ENTER** — выстрел
- Пули летят строго прямо (без тангенциальной скорости)
- Карта **1200×750 px**, полностью видна в окне, симметричная арена

## Быстрый старт

```bash
# Установи зависимости (из корня репо)
pip install -r requirements.txt

# Обучение агента (self-play)
python train.py

# Посмотреть матч
python play.py ai bot      # RL агент vs скриптовый бот (по умолчанию)
python play.py human bot   # ты vs скриптовый бот
python play.py human ai    # ты vs RL агент
python play.py ai ai       # RL агент vs RL агент
```

**Управление (human-режим):**
- `ПРОБЕЛ` (удерживать) — двигаться вперёд
- `ENTER` — выстрел
- `ESC` — выход

## Self-Play

Каждые **200 000 шагов** бот получает копию текущих весов агента и начинает играть умнее. Агент вынужден адаптироваться → цикл повторяется. Так работают AlphaGo, OpenAI Five, AlphaStar.

```
Шаг 0      → бот = скриптовый AI
Шаг 200k   → бот = агент v1
Шаг 400k   → бот = агент v2  (умнее)
Шаг 600k   → бот = агент v3  (ещё умнее)
...
```

## Как устроено

### Observation (32 числа)
| Группа | Что | Размер |
|--------|-----|--------|
| Лучи | 8 лучей до стен | 8 |
| Агент | угол, состояние, скорость, HP, кулдаун | 7 |
| Противник | направление, расстояние, куда смотрит, HP, скорость | 9 |
| Угроза | входящая пуля: риск, время, направление, сторона ухода | 5 |
| Прицел | ошибка упреждения, линия огня | 3 |

### Actions (3 дискретных)
| # | Действие |
|---|----------|
| 0 | Ничего (крутится) |
| 1 | Двигаться вперёд (ПРОБЕЛ) |
| 2 | Выстрел (ENTER) |

### Reward
| Событие | Reward |
|---------|--------|
| Попал в бота | +6 |
| Убил бота | +14 (+18 бонус победы) |
| Получил урон | −5 |
| Умер | −16 (−18 штраф поражения) |
| Уклонился от пули | до +1.8 |
| Качество выстрела | −0.35 … +1.1 |
| Каждый шаг | −0.005 |

### Бот (скриптовый + RL-копия)
Скриптовый бот:
- Рассчитывает упреждение (куда бот **будет** когда пуля долетит)
- Проверяет линию огня перед выстрелом
- Флангует через стены если нет прострела

После каждого обновления бот получает копию RL-политики агента.

## Настройка обучения

В `train.py`:
```python
TOTAL_TIMESTEPS      = 10_000_000  # шагов (больше = умнее)
SELFPLAY_UPDATE_FREQ = 200_000     # как часто обновлять бота
RESUME_TRAINING      = True        # продолжить с сохранённой модели
```
