# RL Car Racing

2D-гоночный симулятор с F1-трассой. Машинка обучается ехать по трассе через Reinforcement Learning (PPO).

![Python](https://img.shields.io/badge/Python-3.10+-blue) ![pygame](https://img.shields.io/badge/pygame-2.x-green) ![SB3](https://img.shields.io/badge/stable--baselines3-PPO-orange)

## Быстрый старт

```bash
# Установи зависимости (из корня репо)
pip install -r requirements.txt

# Ручной режим — поездить самому
python main.py

# Обучение агента
python train.py

# Посмотреть обученную модель
python play_trained.py
```

**Управление в ручном режиме:**
- `↑` — газ
- `↓` — тормоз / задний ход
- `←` / `→` — поворот
- `R` — сброс позиции
- `Esc` — выход

## Как устроено

### Среда
- Мир **3000×2000 px**, камера следует за машинкой
- Трасса — Spa-inspired F1 circuit (39 контрольных точек)
- Генератор случайных трасс через Catmull-Rom сплайны

### Observation (10 чисел)
| # | Что |
|---|-----|
| 0–6 | 7 лучей-сенсоров до края дороги |
| 7 | Нормализованная скорость |
| 8–9 | sin/cos направления машинки |

### Actions (7 дискретных)
`газ` / `газ+влево` / `газ+вправо` / `тормоз` / `влево` / `вправо` / `ничего`

### Reward
| Событие | Reward |
|---------|--------|
| Движение по дороге | +0.1 |
| Близость к центру | до +0.5 |
| Чекпоинт | +2.0 |
| Круг завершён | +5.0 |
| Выезд с дороги | −5.0 |
| Простой | −0.05 |

## Структура

```
car_racing/
├── main.py          — ручной режим
├── train.py         — обучение PPO
├── play_trained.py  — просмотр модели
├── car_env.py       — Gymnasium-среда
├── car.py           — физика + лучи
├── track.py         — F1-трасса + генератор
├── utils.py         — UI утилиты
└── models/          — сохранённые модели (не в git)
    └── ppo_car_racing.zip
```

## Настройка обучения

В `train.py`:
```python
TOTAL_TIMESTEPS = 2_000_000  # шагов обучения
```

В `car_env.py` можно менять reward, длину эпизода, число лучей.
